# 编辑要求
## 目的
最终效果是，以场地类的卡为基准，场地之间的连线构成了精灵界。场地卡的其它连线包括魔法卡、陷阱卡组成的历史节点，怪兽卡代表的人物节点。
人物与人物的连线是这个人物的成长史、关系。
历史节点和历史节点的连线是一段历史的发展，连续的剧情但地点不同，就在新的地点起一条故事线。
历史节点和人物的连线是这个历史包含的任务。
还有些物品卡，在历史节点和人物节点提到链接上即可。

## 步骤

- 将同一个故事的卡移动到同一个文件夹；
- 先对没一张卡分析卡图内容和文本内容，将包含的要素一一列出。
- 选择1张卡作为故事主角的本体。有些对象本体不是怪兽卡。
- 分析情节：卡图中往往包含同一对象。可以看该对象被那些卡片包含，建立一个针对这个对象的小故事。
- 分析小故事之间的关联性，确定故事先后顺序。
- 链接过程中标记卡的类型、标记只有1个：优先标记世界、其次历史。
- 卡片的关联填写中，优先填情节，即卡图、文本中的对象。如果对象间相互包含就选只填一个。
- 其次填情节。
- 其次看情况是否填地点，如果和上一张卡一样就省略。
